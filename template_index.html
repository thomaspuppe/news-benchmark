Infos zum News-Benchmark

Dieses Projekt ist sehr jung, und deswegen noch sehr spartanisch. Sowohl die getesteten Seiten als auch die bewerteten Metriken/Scores werden in Zukunft erweitert.

Der Code sowie die Messergebnisse sind Open Source und auf GitHub zu finden: https://github.com/thomaspuppe/news-benchmark.

Disclaimer: Der Autor dieses Projekts ist Mitarbeiter von ZEIT Online.

Welche Seiten werden getestet?

In der ersten Version nur fünf: ZEIT Online, WELT, Spiegel, FAZ und die Süddeutsche Zeitung.

Wie wird getestet?

Fürs Erste wird das Tool Lighthouse von Google Chrome genutzt. Es testet Websites und erstellt Scores für die Themen Accessibility, Performance, Best Practices und SEO. Aus diesen Scores wird das Ranking erstellt.

Da kann man eine Menge untersuchen, erklären, und verbessern. Zum Beispiel erhalten die ZEIT und die WELT die schlechtesten Performance-Scores, obwohl die Seiten objektiv kleiner und subjektiv schneller sind als die anderen.

Again: das ist die erste Version. Feedback via Twitter und E-Mail ist herzlich willkommen!
